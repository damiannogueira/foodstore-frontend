# 🛒 FoodStore Frontend

Frontend desarrollado en TypeScript, HTML y CSS para la gestión de una tienda online.

Se conecta con el backend mediante API REST y permite la interacción completa tanto de usuarios clientes como administradores.

---

## 🚀 Tecnologías utilizadas
* TypeScript
* HTML5
* CSS3 (estilos reutilizables)
* Vite
* Fetch API
* LocalStorage

---

---

## 📦 Funcionalidades

### 👤 Cliente
* Registro e inicio de sesión
* Visualización de productos
* Filtro por categoría
* Búsqueda y ordenamiento
* Detalle de producto
* Carrito de compras
* Creación de pedidos
* Historial de pedidos

### 🛠️ Administrador
* Dashboard con resumen
* Gestión de categorías (CRUD)
* Gestión de productos (CRUD)
* Gestión de pedidos

---
